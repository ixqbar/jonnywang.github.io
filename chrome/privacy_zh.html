隐私政策

最后更新日期：2025年3月

1. 概述

本隐私政策适用于 [扩展名称] Chrome 扩展（以下简称“扩展”）。我们非常重视您的隐私，并承诺不会收集、存储或上传您的任何个人数据或浏览数据。

2. 数据收集与使用

本扩展不会收集、存储或上传以下任何数据：

  • 您的个人身份信息（如姓名、电子邮件地址等）。
  • 您的浏览历史记录或网站访问数据。
  • 您在网页上输入的任何内容。

扩展的所有功能均在您的浏览器本地运行，不会将任何数据传输到外部服务器。

3. 扩展功能说明

本扩展提供以下功能：

  • 展示原始页面素材库图片，提供复制按钮操作： 扩展会读取当前页面中的图片素材库，并提供复制按钮以便您快速复制图片链接或资源。此操作仅在您的浏览器本地完成，不会上传任何数据。
  • 通过页面官方提供 API 对草稿编辑器进行直接插入、提取 HTML 操作： 扩展会与页面官方 API 交互，以支持在草稿编辑器中插入或提取 HTML 内容。此操作仅在您的浏览器本地完成，不会上传任何数据。
  • 展示原始页面关注用户列表，提供页面元素属性 fakeid 的按钮复制操作： 扩展会读取当前页面中的关注用户列表，并提供复制 fakeid 的按钮。此操作仅在您的浏览器本地完成，不会上传任何数据。

4. 权限说明

为了实现上述功能，扩展需要以下权限：

  • 访问特定网站： 扩展需要访问您使用的特定网站，以便读取页面内容并提供相关功能。扩展不会访问与功能无关的网站。
  • 读取和修改页面内容： 扩展需要读取页面内容以展示素材库图片、关注用户列表等，并可能修改页面内容以支持草稿编辑器的操作。
所有权限仅用于实现扩展功能，不会用于其他用途。

5. 数据安全

由于扩展不会收集或上传任何数据，因此不存在数据存储或传输的安全风险。所有功能均在您的浏览器本地运行，确保您的隐私得到最大程度的保护。
