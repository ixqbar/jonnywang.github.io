fcgiproxy

星期八的博客
2017-12-28
websocket php 长连接 redis

项目地址

  • https://github.com/jonnywang/fcgiProxy

用途

  • 客户端使用websoket与fcgiproxy建立长链接，数据经由fcgiproxy转发给php-fpm,php服务端接收数据并处理原路返回给客户端，php客户端也可以使用fcigproxy内置的redis协议把要推送的数据直接推送给指定客户端。
  • 利用内置redis协议推送消息给iOS,android,Desktop用于提醒相关事项。
